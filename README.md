# DevConnect Backend II API

Backend desarrollado como proyecto académico utilizando Node.js, Express.js y MongoDB.

## 📚 Tecnologías utilizadas

* Node.js
* Express.js
* MongoDB
* Mongoose
* Git
* GitHub

## ✅ Funcionalidades implementadas

* Crear usuarios
* Obtener todos los los usuarios
* Obtener usuario por ID
* Actualizar usuario
* Eliminar usuario
* Validación de email existente

## 📂 Estructura del proyecto

```
src/
├── controllers
├── models
├── routes
├── config
├── app.js
└── server.js
```

## ⚙️ Instalación

```bash
npm install
```

## ▶️ Ejecutar el proyecto

```bash
npm run dev
```

## 🌐 API Base

```
http://localhost:3000/api/v1
```
